# CovidDataDashboard

### Description
A replica of the ministry of health's covid data dashboard from 2020, it includes:
- REST API (Node.JS and Express) 
- Graphs

### Technologies
- Vanilla Javascript
- Node.JS + Express
- Highcharts
- Chartjs
- MongoDB
- Sass
